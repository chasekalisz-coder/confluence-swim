progression. Update the codebase from this file; never the other way around.

Course order: all SCY events first, then all LCM events.
Stroke order within each course: Freestyle → Butterfly → Backstroke → Breaststroke → Individual Medley.
Event order within each stroke: shortest → longest.

All swims are included — prelims and finals both count. "X" markers from USA
Swimming source are dropped since they don't affect legitimacy.

Rule: if the athlete's profile-listed best time is faster than anything in
the 10 uploaded results, that best time is added to the list (labeled
`pending` for date and meet) and the slowest of 10 is dropped. If the list
has fewer than 10, the profile best is simply appended and nothing is removed.
